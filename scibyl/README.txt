The codes available here are written for Python 3.4 or later. 

Required Libraries:
------------------
All required libraries are available via the Python Package Index ("pip"):
    - matplotlib
    - numpy
    - theano

Recommendations:
---------------
Install the following distribution, which comes with most of the above:
    - Anaconda (http://continuum.io/downloads#py34)

You will still need to install pygep and theano by hand.
